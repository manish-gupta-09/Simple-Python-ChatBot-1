# 🤖 Simple Python Chatbot

A simple rule-based chatbot developed in Python that processes user input using string processing, conditionals, and dictionaries. The chatbot recognizes different types of user queries and responds accordingly.

---

## 📌 Features

- 👋 Greeting Detection
- 😊 "How are you?" Responses
- 🆔 Introduces Itself
- ⏰ Displays Current Time
- 🌦 Weather-related Responses
- 😂 Tells Random Jokes
- 🙏 Responds to Thanks
- 📖 Help Command
- 💬 Small Talk
- 🚪 Exit Command
- ❓ Handles Unknown Questions
- 🚫 Handles Empty Input

---

## 🛠 Technologies Used

- Python 3.x
- random module
- datetime module

## 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/yourusername/Simple-Python-Chatbot.git
```

Move inside the project

```bash
cd Simple-Python-Chatbot
```

Run the chatbot

```bash
python simple_chatbot.py
```
